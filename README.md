## Great-Productivity-Tools

A simple list of AI tools I use for productivity, categorized by scenarios.

### Frontend UI
- **Stitch**: AI-native design canvas and prototyping tool from Google Labs that generates high-fidelity app and website user interfaces and corresponding code from natural language. [Website](https://stitch.withgoogle.com/)

### General Coding Agent
- **Claude Code**: Agentic CLI tool by Anthropic that acts as an AI coding assistant directly within your terminal. [Website](https://claude.com/product/claude-code)
- **Codex**: Coding agent by OpenAI. [Website](https://chatgpt.com/codex)


### General LLM
- **Grok**:  AI assistant developed by xAI. [Website](https://grok.com/)
- **ChatGPT**: AI assistant developed by OpenAI. [Website](https://chatgpt.com/)
- **Claude**: AI assistant developed by Claude. [Website](https://claude.ai/)
- **Gemini**: AI assistant developed by Google. [Website](https://gemini.google.com/app)
- **Qwen**: AI assistant developed by Alibaba. [Website](https://qwen.ai/home)
- **DeepSeek**: AI assistant developed by DeepSeek AI. [Website](https://www.deepseek.com/)
- **Kimi**: AI assistant developed by Moonshot AI. [Website](https://www.kimi.com/)


### Study/Work
- **Notion**: Centralized workspace for notes and project tracking from Notion Labs. [Website](https://www.notion.com/)
- **NotebookLM**: Upload your documents and output summaries, Q&A, audio overviews, and concept maps from Google Labs. [Website](https://notebooklm.google/)
